# Awesome Math Visual Atlas

A visual, searchable static website for [`rossant/awesome-math`](https://github.com/rossant/awesome-math).

## What it does

- Reads the original Awesome Math README from GitHub.
- Converts links into resource cards.
- Supports keyword search and resource type filtering.
- Shows a visual math branch map.
- Works as a simple GitHub Pages static site.

## Files

```text
index.html
README.md
.nojekyll
```

## GitHub Pages

Recommended Pages settings:

```text
Source: Deploy from a branch
Branch: main
Folder: /root
```

After GitHub Pages is enabled, the public site should be available at:

```text
https://ocean-Go.github.io/awesome-math/
```
